# ADR-0004 — Costeo: Promedio ponderado
**Estado:** ACEPTADO (no editar; nuevos ADR para cambios)  
**Fecha:** 2026-02-09  

---

## Contexto
Se requiere un método de costeo estable y simple para reportes de margen.

## Decisión
- Método global: promedio ponderado (weighted average).

## Consecuencias
- Margen consistente con recepciones parciales.
- Implementación más simple que FIFO para este producto.
